<h1>DESENVOLVIMENTO JAVA WEB</h1>

<p>Trabalho proposto pelo Profº Bruno Zolotareff dos Santos, do curso de Ciência da Computação na FMU, 
   para desenvolver um sistema web de tema livre(algo com o qual se identifica) com páginas JSP usando 
   Servlets, scriptlets, Session e JDBC. 
   Surgiu então a idéia de fazer um site de consultoria em desenvolvimento de software. O diferencial
   é que para isto utilizei os frameworks Hibernate, Spring MVC e Bootstrap.
</p>
